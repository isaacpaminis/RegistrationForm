Registration Form : 

This a simple registration form with real-time form validation using HTML, CSS, and JavaScript. The form includes fields for Name, Email, and Password, and validates user input dynamically before submission. 

Error messages are displayed when the input does not meet the required criteria, ensuring accurate data entry.

The Submit button remains disabled until all fields are properly validated, demonstrating practical use of client-side validation, DOM manipulation, and event handling in JavaScript.

Tools Used :

HTML5

CSS3

JavaScript

Features: 

Real-time form validation

Name field validation (cannot be empty)

Email format validation

Password minimum length validation (6 characters)

Dynamic error messages for invalid inputs

Submit button enabled only when all fields are valid
